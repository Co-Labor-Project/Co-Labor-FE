## Installation & Run

**1. Run the following commands in your desired directory.**  
```bash
git clone  
cd Co-Labor  
```

**2. Download Docker images**
```bash
docker compose pull  
```

**3. Start Docker services**
```bash
docker compose up -d  
```

### Access
- Frontend: http://localhost:5173
- Webpage: https://colabor.site/
